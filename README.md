# Aura: Universal Sovereign Rollup Framework 🛡️

The open standard for AI-guarded, modular L3 sovereign rollups. Built for the era of hyper-scale and decentralized intelligence (Vision 2026).

## 📂 Monorepo Structure

- **[aura-l3-contracts/](./aura-l3-contracts/)**: Core Smart Contracts (Factory, Guardian Oracle) using Hardhat.
- **[aura-guardian-node/](./aura-guardian-node/)**: AI-driven node that audits L3 configurations off-chain.
- **[aura-cli/](./aura-cli/)**: Developer CLI tool (`aura-l3`) for creating and deploying rollups.
- **[docs/](./docs/)**: Technical specifications and strategic framework.
- **[aura-wallet-ledger/](./aura-wallet-ledger/)**: Sovereign ledger infrastructure and mobile/web wallet core.

## 🚀 Quick Start (Development)

1. **Install Dependencies**: `npm install` in each sub-directory.
2. **Setup AI Guardian**: Navigate to `aura-guardian-node` and run `npm start`.
3. **Deploy L3 via CLI**: Use `npm install -g ./aura-cli` and run `aura-l3 create`.
